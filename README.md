# ✨ PromptCraft AI – Smart Prompt Generator

PromptCraft AI is a modern full-stack AI web application that generates optimized prompts using Google Gemini API. Users provide task details such as task type, topic, tone, and output format, and the application returns:
- Optimized Prompt
- Why the Prompt Works
- Expert-Level Prompt
- One-Click Copy to Clipboard

---

## 🚀 Features

- AI-powered prompt generation using Google Gemini (`gemini-2.5-flash`)
- Prompt engineering explanations
- Expert-level prompt enhancements
- Responsive glassmorphism UI
- Secure API key storage with environment variables
- Copy-to-clipboard functionality

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- AI Model: Google Gemini API
- Deployment: Netlify (Frontend) + Render (Backend)

## ⚙️ Setup Instructions

### 1. Clone Repository
git clone https://github.com/meshramshreya28-code/PromptCraft-AI.git

### 2. Install Backend Dependencies
cd PromptCraft-AI/backend
npm install

### 3. Create `.env`
GEMINI_API_KEY=your_api_key_here
PORT=5000

### 4. Run Backend
npm run dev

### 5. Open Frontend
Open `frontend/index.html` in your browser or use Live Server.
